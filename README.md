# Premium Coffee Shop Landing Page

A modern, high-end landing page for an exclusive coffee shop experience. Built with React, Vite, and TypeScript, featuring luxury branding, responsive design, and transaction flow integration.

## Features

- **Premium Design**: Sophisticated dark theme with luxury coffee shop aesthetics
- **Hero Section**: Compelling call-to-action with high-end visuals
- **Transaction Flow**: Integrated order, reservation, and gift card sections
- **Responsive Layout**: Mobile-first design that works on all devices
- **Business Statistics**: Key metrics and performance indicators
- **Feature Showcase**: Highlighting unique offerings and services
- **Docker Ready**: Fully containerized for easy deployment

## Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: CSS3 with responsive media queries
- **Deployment**: Docker & Docker Compose
- **Runtime**: Node.js

## Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- Docker & Docker Compose (optional, for containerized deployment)
